# Psychologically-approved colour palette:
**Primary (Teal)**
- #0C8C8C — Primary action, buttons, links
- #0A7373 — Hover/active state
- #085C5C — Pressed/dark emphasis
- #E8F6F6 — Soft tint (selected rows, active nav backgrounds)
- #D0EEEE — Medium tint (badges, chips, subtle highlights)

**Secondary (Navy)**
- #1A2E44 — Headings, high-emphasis text
- #2A4365 — Subheadings, secondary emphasis

**Background**
- #FAFAF8 — Page background (that warm white I mentioned)
- #FFFFFF — Card / surface
- #F5F4F0 — Recessed / muted sections (sidebar, table headers)
- #EFEEE9 — Inset / well (code blocks, data panels)

**Borders**
- #E6E4DF — Default card/divider border
- #D4D1CA — Stronger borders (active cards, table lines)
- #C1E8E8 — Primary-tinted border (selected card, focused input)

**Text**
- #1A2E44 — Primary text
- #5A6578 — Secondary / muted text
- #8C919C — Placeholder / disabled text

**Semantic**
- #0B7A5E — Success
- #C27A15 — Warning
- #C93B3B — Error / critical

The warm neutrals in the backgrounds and borders are what'll make it feel human rather than sterile. And notice the semantic colors are slightly desaturated compared to what you'd normally see — that's intentional so they don't visually scream against the calm teal palette.
